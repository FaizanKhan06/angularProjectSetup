## Running the slides locally
- Clone this repository
- `npm install`
- `npm run dev`


## Running the app

To run the app we built in the video tutorial (the final state):
- Clone this repository if you haven't
- `cd first-ng-app`
- `npm install`
- `npm start`
- Navigate to [localhost:4200](http://localhost:4200)
